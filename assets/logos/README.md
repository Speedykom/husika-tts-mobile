## Before publishing — verify against the visibility rules

These are best-available official files; confirm them with the PRBP / GIZ
communications team before any public release:

- **EU emblem:** legally binding rules apply. The "Co-funded by the European
  Union" wording must stay next to the emblem, with minimum size and clear space.
  Authoritative source / all language versions:
  https://ec.europa.eu/regional_policy/information-sources/logo-download-center_en
- **German Cooperation:** the project may require the co-branding lockup
  "German Cooperation – Deutsche Zusammenarbeit" (from the GIZ Corporate Design
  Centre) instead of the plain BMZ ministry logo used here.
- **ICPAC:** replace with ICPAC's official light-background logo if one is provided.
- Keep ordering and relative sizing consistent with the donor agreement
  (funders → implementer → partners → developer).
